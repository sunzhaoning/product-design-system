# 产品设计知识库

> 本目录是产品设计辅助系统的知识积累中心，所有经过验证的设计原则、交互模式、文案规范均沉淀于此。

## 目录结构

| 文件/目录 | 用途 | 更新频率 |
|-----------|------|----------|
| `01-原型设计规范.md` | 原型技术路线与视觉规范 | 按需（技术决策变更时） |
| `02-设计原则库.md` | 从实际案例提炼的可复用设计原则 | 每次需求复盘后 |
| `03-交互模式库.md` | 从实际需求提炼的交互流程模式 | 每次需求复盘后 |
| `04-文案规范.md` | 界面文案与提示语规范 | 按需 |
| `05-业务术语表.md` | 产品领域专业词汇与概念定义 | 按需 |
| `component-templates/` | HTML 原型模板 | 按需 |
| `design-decisions/` | 设计决策记录（ADR） | 重要决策时 |
| `retrospectives/` | 复盘日记 | 每个需求完成后 |

## 知识治理规则

1. **准入条件**：每条知识条目必须引用至少 1 个实际案例；能提升下一次生成质量
2. **更新方式**：半自动——系统建议候选条目，人工确认后写入
3. **冲突处理**：产品专属规则 > 通用规则 > 默认值
4. **知识生命周期**：条目可标记为"已废弃"，不删除历史记录

## 快速入口

- 新建需求原型 → 参考 `01-原型设计规范.md` + `component-templates/`
- 复盘提炼 → 使用 `retrospectives/_template.md`
- 记录设计决策 → 使用 `design-decisions/_template.md`
