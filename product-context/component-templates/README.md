# 组件模板使用说明

> 本目录包含可直接复用的 HTML 原型基础模板。每个模板均为单文件（内联 CSS + JS），无需外部依赖，开箱即用。

## 可用模板

| 文件 | 适用场景 |
|------|----------|
| `table-page.html` | 标准列表页（带搜索、筛选、批量操作、分页） |

## 使用方式

1. 复制目标模板到新需求文件夹，重命名为需求对应名称（如 `用户管理列表.html`）；
2. 修改页面标题（`<title>` 和页头标题文字）；
3. 按需调整表头列、筛选条件、操作按钮等占位内容；
4. 遵循 `product-context/01-原型设计规范.md` 中的视觉原则。

## 如何添加新模板

参考现有模板的结构约定：
- 顶部 `/* ===== CSS Variables ===== */` 区块集中管理设计变量；
- HTML 注释标注每个可自定义区域（如 `<!-- 自定义：修改表头列定义 -->`）；
- JS 逻辑仅处理原型必要的交互（全选、分页切换等），不引入框架。
