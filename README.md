# 产品设计辅助系统

这是一个自进化的产品设计辅助系统，用于协助产品经理完成交互原型搭建、需求文档编写以及经验复盘提炼。

```

├── product-context/          # 知识库
│   ├── 01-原型设计规范.md
│   ├── 02-设计原则库.md
│   ├── 03-交互模式库.md
│   ├── 04-文案规范.md

│   ├── component-templates/
│   ├── design-decisions/
│   └── retrospectives/
└── README.md
```
